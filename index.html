FOR AUTO ESCALATION[Issue ID: 1-IGB7IV][POSSIBLE HOT ISSUE][TMMS/iTMMS] Unable to activate TMMS/iTMMS | Unable to continue - Error code #98000006 | #98000004
Case Management
Field Value
SR Title: SUnable to activate TMMS/iTMMS | Unable to continue - Error code #98000006 | #98000004
Product: TREND MICRO MOBILE SECURITY
Issue ID: 1-HXW9K1
Problem Category: Registration
Sub-Category: Registration Issue
Problem
  • Cannot activate Trend Micro Mobile Security on android or iOS device.
  • Unable to continue - Error code #98000006 or #98000004 when trying to activate the program.
Solution
  • For error code #98000004
    1. Check with client if she did a fresh installation/upgrade of the program.
    2. Check with client if she recently did a sign out/in on the program.
    3. Collect the needed information below.
      • Email address used to activate.
      • Serial number
    4. Collect iTMMS/TMMS logs (Note: Make sure to replicate the issue and note the TIMESTAMP.
    5. Escalate to L2.
  • For error code #98000006
    1. Follow the steps on the link here.
    2. If the issue persists:
      • Check with client if she did a fresh installation/upgrade of the program.
      • Check with client if she recently did a sign out/in on the program.
    3. Collect the needed information below
      • Email address used to activate.
      • Serial number
    4. Provide spiel:
      • “We are already aware of this issue and is currently undergoing investigation. Rest assured that we’ll provide updates once available.”

  2. Proceed with the troubleshooting
  3. Submit URL reclassification, then report to Tech Support Scam URL online tracker
  4. Tag Tech Support Scam cases under Issue ID: 1-H7GOK0 For addition information, visit: Frequently Asked Questions (FAQ) about Suspected Third Party Technical Support Scams
